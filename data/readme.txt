#This directory contains all the data ( ".mat" - MATLAB format) required to generate all the figures for the paper. 
Each matrix represents microstructural data (R1) from infants per region of interest (ROIs) across  
three visual streams (dorsal/lateral/ventral) 
tissue types (gray/white matter: GM/WM) and 
hemispheres (left/right hemisphere: LH/RH) 

Each matrix contains 5 structures, including:
1. infant's age in days
2. mean R1 values per ROIs and infant
3. infant session list
4. hemisphere (left/right)
5. ROI list 

ROIs list is as follows: 

Early visual  - V1 (V1v + V1d), V2 (V2v + V2d), and V3 (V3v + V3d) (these 3 regions are combined with each visual stream)

Ventral: (V1 V2 V3) + hV4 V01 V02 PHC1 PHC2

Dorsal:   (V1 V2 V3) +  V3a V3b IPS0 IPS1 IPS2

Lateral: (V1 V2 V3) + LO1 LO2 TO1 TO2 MTG
